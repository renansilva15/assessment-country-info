# Country Info

## Requirements

- [Node.js](https://nodejs.org/en)

## Execute

### Setup the environment
```bash
cp .env.example .env
```

### Install the dependencies
```bash
npm install
```

### Start the application
```bash
npm run start
```

To see if it is working go to http://localhost:3333/api-docs
